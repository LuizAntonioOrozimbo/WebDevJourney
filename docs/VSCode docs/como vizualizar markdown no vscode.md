Para visualizar um arquivo Markdown no VS Code, você pode usar o atalho de teclado Ctrl + Shift + V (no Windows/Linux) ou Cmd + Shift + V (no Mac) para abrir a pré-visualização lado a lado, ou Ctrl + K + V (Windows/Linux) ou Cmd + K + V (Mac) para abrir em uma aba separada. Também é possível clicar no ícone de pré-visualização (geralmente um ícone de documento no canto superior direito do editor) para ver a renderização do arquivo. 

Para visualizar um arquivo Markdown no VS Code, você pode usar o atalho de teclado Ctrl + Shift + V (no Windows/Linux) ou Cmd + Shift + V (no Mac) para abrir a pré-visualização lado a lado, ou Ctrl + K + V (Windows/Linux) ou Cmd + K + V (Mac) para abrir em uma aba separada. Também é possível clicar no ícone de pré-visualização (geralmente um ícone de documento no canto superior direito do editor) para ver a renderização do arquivo. 
Opções de visualização
Pré-visualização lado a lado: Pressione Ctrl + Shift + V (Windows/Linux) ou Cmd + Shift + V (Mac) para abrir o editor com a pré-visualização em tempo real ao lado do código.
Pré-visualização em aba separada: Use o atalho Ctrl + K + V (Windows/Linux) ou Cmd + K + V (Mac) para abrir a visualização em uma nova aba do editor.
Ícone de pré-visualização: Clique no ícone de pré-visualização (geralmente um ícone de documento com uma seta ou um olho) no canto superior direito da janela do editor para alternar entre o modo de edição e o modo de visualização.
Menu de contexto: Clique com o botão direito do mouse no arquivo Markdown na barra lateral do Explorer para ver a opção "Open Preview" (Abrir visualização) no menu de contexto.
